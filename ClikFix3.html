Tutorial Menjalankan Script di PowerShell

  1. Tekan tombol Windows + R.
  2. Ketik powershell lalu tekan Enter.
  3. Copy script di atas (blok dan tekan Ctrl + C).
  4. Paste ke PowerShell (klik kanan atau Ctrl + V).
  5. Tekan Enter untuk menjalankan script.
